# Calculadora-Python

Este repositório documenta o desenvolvimento progressivo de diferentes versões de uma calculadora em Python.
O objetivo é estudar e aplicar conceitos relacionados a lógica de cálculo, programação orientada a objetos e interfaces gráficas, utilizando ferramentas como Tkinter.

Cada versão do projeto representa uma etapa de aprendizado, incorporando novas funcionalidades e melhorias na estrutura do código.
